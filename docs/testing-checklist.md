# WebJuggler 테스트 체크리스트

## 기본 기능

### 인증
- [ ] 회원가입 (Register 버튼)
- [ ] 로그인 (Login 버튼)
- [ ] 로그아웃 (Logout 버튼)
- [ ] 토큰 만료 시 자동 로그아웃 + 토스트 알림

### 파일 관리
- [ ] ULG 파일 업로드 (Upload 버튼)
- [ ] 여러 파일 동시 로딩 (Upload 여러 번)
- [ ] 사이드바에 파일별 그룹 표시
- [ ] 파일 옆 × 버튼으로 언로드
- [ ] 잘못된 파일 업로드 시 에러 토스트 표시

### 사이드바
- [ ] 토픽 트리 펼치기/접기 (토픽명 클릭)
- [ ] 필드 필터 검색 (Filter fields... 입력)
- [ ] 사이드바 접기/펼치기 (◀/▶ 버튼)
- [ ] 다중 파일 시 파일별 그룹 접기/펼치기

---

## 플롯

### 드래그 & 드롭
- [ ] 필드 1개 드래그 → 타임시리즈 플롯
- [ ] Ctrl+클릭으로 필드 2개 선택 후 드래그 → X-Y 플롯
- [ ] Ctrl+클릭으로 필드 3개 선택 후 드래그 → 3D 스캐터 플롯
- [ ] 기존 타임시리즈 플롯에 필드 추가 드래그 → 다중 시리즈
- [ ] 다른 파일의 같은 필드를 하나의 플롯에 드래그 → 비교

### 타임시리즈 플롯
- [ ] 마우스 드래그로 줌 (영역 선택)
- [ ] 더블클릭으로 줌 리셋
- [ ] 마우스 휠로 줌 인/아웃
- [ ] 줌/팬이 모든 타임시리즈 플롯에 동기화됨

### X-Y 플롯
- [ ] 시간순 궤적 선 + 점 표시
- [ ] 커서 동기화 시 해당 시점 하이라이트 (흰 원)

### 3D 플롯
- [ ] 마우스 드래그로 회전
- [ ] 마우스 휠로 줌
- [ ] 궤적 선 + 축 라벨 표시
- [ ] 커서 동기화 시 흰 구체가 해당 시점 따라감

### 범례 (Legend)
- [ ] 플롯 우상단에 범례 표시 (색상 + 필드명 + 현재값)
- [ ] 호버 시 실시간 수치 업데이트
- [ ] 범례 항목 클릭 → 시리즈 숨기기/보이기 토글
- [ ] 범례 항목 우클릭 → 시리즈 제거

### 커서 동기화
- [ ] 한 플롯에서 호버 → 다른 플롯에 수직 커서선 표시
- [ ] X-Y 플롯에서 해당 시점 하이라이트
- [ ] 3D 플롯에서 해당 시점에 흰 구체 표시

## 레이아웃

### 분할
- [ ] 우클릭 → Split Vertical (세로 분할)
- [ ] 우클릭 → Split Horizontal (가로 분할)
- [ ] 분할 경계선 드래그로 비율 조절
- [ ] 우클릭 → Close Panel (패널 닫기)
- [ ] 우클릭 → Clear Series (시리즈 클리어)

### 저장/복원
- [ ] 브라우저 새로고침 후 레이아웃 유지
- [ ] 새로고침 후 시리즈 데이터 자동 재요청
- [ ] 새로고침 후 로드된 파일 목록 유지

## 재생 컨트롤

- [ ] ▶ 버튼으로 재생 시작
- [ ] ⏸ 버튼으로 일시정지
- [ ] 타임라인 슬라이더 드래그로 시점 이동
- [ ] ◀◀ / ▶▶ 버튼으로 스텝 이동
- [ ] 속도 변경 (0.5x, 1x, 2x, 5x, 10x)
- [ ] 재생 중 커서가 모든 플롯에서 이동
- [ ] 끝까지 재생 시 처음으로 루프

## 키보드 단축키

| 단축키 | 동작 | 조건 |
|--------|------|------|
| **V** | 세로 분할 (Split Vertical) | 플롯 포커스 필요 (클릭) |
| **H** | 가로 분할 (Split Horizontal) | 플롯 포커스 필요 |
| **Delete** / **Backspace** | 시리즈 클리어 | 플롯 포커스 필요 |
| **Ctrl+Z** | 실행 취소 (Undo) | 항상 |
| **Ctrl+Y** / **Ctrl+Shift+Z** | 다시 실행 (Redo) | 항상 |
| **Space** | 재생/일시정지 토글 | 입력 필드 미포커스 |
| **Esc** | 컨텍스트 메뉴 닫기 | 메뉴 열려있을 때 |

## 알려진 제한사항

- 500MB 이상 파일 업로드 불가 (서버 설정)
- 파싱 실패하는 ulg 파일이 있을 수 있음 (에러 토스트로 표시)
- X-Y / 3D 플롯은 줌 동기화 대상 아님 (타임시리즈만)
- 데이터 변환 (미분, 이동평균 등) 미구현
