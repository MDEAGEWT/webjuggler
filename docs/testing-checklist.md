# WebJuggler 테스트 체크리스트

## 기본 기능

### 인증
- [ ] 회원가입 (Register 버튼)
- [ ] 로그인 (Login 버튼)
- [ ] 로그아웃 (Logout 버튼)
- [ ] 토큰 만료 시 자동 로그아웃 + 토스트 알림

### 파일 관리
- [ ] ULG 파일 업로드 (Upload 버튼)
- [ ] 여러 파일 동시 로딩 (Upload 여러 번)
- [ ] 사이드바에 파일별 그룹 표시
- [ ] 파일 옆 × 버튼으로 언로드
- [ ] 잘못된 파일 업로드 시 에러 토스트 표시

### 사이드바
- [ ] 토픽 트리 펼치기/접기 (토픽명 클릭)
- [ ] 필드 필터 검색 (Filter fields... 입력)
- [ ] 사이드바 접기/펼치기 (◀/▶ 버튼)
- [ ] 다중 파일 시 파일별 그룹 접기/펼치기
- [ ] 긴 토픽명 ellipsis 처리
- [ ] 커서 위치의 필드값 표시 (플롯에 추가된 필드만)

## 플롯

### 드래그 & 드롭
- [ ] 필드 1개 드래그 → 타임시리즈 플롯
- [ ] Ctrl+클릭으로 필드 2개 선택 후 드래그 → X-Y 플롯
- [ ] Ctrl+클릭으로 필드 3개 선택 후 드래그 → 3D 스캐터 플롯
- [ ] Ctrl+클릭으로 필드 4개 선택 후 드래그 → Attitude 뷰
- [ ] 기존 타임시리즈 플롯에 필드 추가 드래그 → 다중 시리즈
- [ ] 다른 파일의 같은 필드를 하나의 플롯에 드래그 → 비교
- [ ] 다른 토픽의 필드도 같은 플롯에서 동시 표시 (타임스탬프 병합)

### 타임시리즈 플롯
- [ ] X축 드래그 줌 (영역 선택)
- [ ] X+Y 드래그 줌 (영역 선택으로 Y축도 줌)
- [ ] 더블클릭으로 줌 리셋
- [ ] 줌/팬이 모든 타임시리즈 플롯에 동기화됨 (Zoom sync ON 시)

### X-Y 플롯
- [ ] 시간순 궤적 선 + 점 표시
- [ ] 커서 동기화 시 해당 시점 하이라이트 (흰 원)
- [ ] 축 반전 버튼 (좌하단 X/Y)

### 3D 플롯
- [ ] 마우스 드래그로 회전
- [ ] 마우스 휠로 줌
- [ ] 궤적 선 + 축 라벨 (X/Y/Z) 표시
- [ ] Z축이 수직 (위/아래)
- [ ] 축 반전 버튼 (좌하단 X/Y/Z)
- [ ] 커서 동기화 시 흰 구체가 해당 시점 따라감

### Attitude 뷰 (쿼터니언)
- [ ] 4필드 드래그 → 3D 자세 표시 (RGB 축 화살표)
- [ ] 8필드 드래그 → 2개 자세 비교 (주황 vs 시안 막대)
- [ ] 커서 시간에 따라 자세 업데이트
- [ ] PX4 NED 좌표계 매핑
- [ ] 오일러 각도 오버레이 (R/P/Y, grid 정렬)
- [ ] 우하단에 X(빨) Y(초) Z(파) 축 범례
- [ ] Q 버튼으로 attitude 모드 전환, T 버튼으로 복귀

### 나침반 뷰
- [ ] 단일 필드 → 나침반 모드 전환 (🧭 버튼)
- [ ] 다중 필드 → 여러 침 표시 (첫 번째=빨강, 나머지=시리즈색)
- [ ] 자동 라디안↔도 변환
- [ ] 하단 오버레이에 토픽/필드 경로 + 값 (grid 정렬)
- [ ] 라이트/다크 모드 모두 정상 표시

### 범례 (우상단)
- [ ] 색상 칩 + 토픽/필드 경로 표시
- [ ] 클릭 → 시리즈 숨기기/보이기 토글
- [ ] 우클릭 → 시리즈 제거
- [ ] 다중 파일 시 [파일명] 접두사

### 커서 값 오버레이 (세로선 옆)
- [ ] 커서 세로선 옆에 시간 + 각 시리즈 값 표시
- [ ] 오른쪽 넘치면 왼쪽으로 전환
- [ ] 각 시리즈 교차점에 색상 점 표시
- [ ] nearest-neighbor로 항상 모든 시리즈 값 표시
- [ ] 소수점 3자리 반올림
- [ ] 커브 Y중간에 수직 위치

### 커서 모드 (우측 사이드바)
- [ ] OFF — 재생바로만 커서 이동, 마우스로 안 움직임
- [ ] Point — 마우스 근처 가장 가까운 커브 데이터 포인트에 점 + 툴팁 (name/time/value)
- [ ] Time — 마우스 호버로 시간 트래커 이동
- [ ] 클릭할 때마다 OFF→Point→Time→OFF 순환

## 키보드 단축키

| 단축키 | 동작 | 조건 |
|--------|------|------|
| **V** | 세로 분할 (Split Vertical) | 플롯 포커스 필요 (클릭) |
| **H** | 가로 분할 (Split Horizontal) | 플롯 포커스 필요 |
| **Delete** / **Backspace** | 시리즈 클리어 | 플롯 포커스 필요 |
| **Ctrl+Z** | 실행 취소 (Undo) | 항상 |
| **Ctrl+Y** / **Ctrl+Shift+Z** | 다시 실행 (Redo) | 항상 |
| **Space** | 재생/일시정지 토글 | 입력 필드 미포커스 |
| **Esc** | 컨텍스트 메뉴 닫기 | 메뉴 열려있을 때 |

## 알려진 제한사항

- 500MB 이상 파일 업로드 불가 (서버 설정)
- 파싱 실패하는 ulg 파일이 있을 수 있음 (에러 토스트로 표시)
- X-Y / 3D 플롯은 줌 동기화 대상 아님 (타임시리즈만)
- WebGL Context Lost 가끔 발생 (Three.js 뷰 많이 열 때)
